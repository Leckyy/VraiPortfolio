Portfolio AURIAT Tristan

Je m'appelle Tristan AURIAT je suis étudiant aux lycée Parc de Vilgenis en deuxième année de SIO (Services Informatiques aux Organisations ) .Grace a mes stages effectuer mes années précedentes j'ai découvert le développement web. Ceci ma permis de découvrir sur différentes formes en le codant, avec wordpress ou même Wix. J'ai beaucoup aimer réaliser des sites internets c'est pour cela que j'ai rejoins le lycee Parc de Vilgenis pour m'approfondir dans ce domaines. Pour cela en premiere année on a des cours pour apprendres le html css et php avec des minis-tp et des projets à réaliser pour ce former d'avantage. En deuxieme années on nous apprends toujours le développement mais cette fois ci plus approfondi

Les langages

Pendant ces 2 années j'ai appris différents langages tel que le HTML, CSS, Javascript, PHP, C# et python

HTML

Le langage HTML est celui qu'on a appris en début d'annee avec le langage CSS en apprenant les base avec des minis-tp. Le langage n'est pas très compliqué

CSS

Le langage CSS est celui qu'on a appris en début d'année avec le langage HTML en apprenant les bases avec des ministp. Pour moi faut connaitre plutôt bien le langage pour créer de beaux sites

Javascript

Malheureusement Javascript est un langage qu'on n'a pas beaucoup approfondi du coup j'ai dû apprendre de moi-même . Je sais faire quelque petite chose comme gérer un formulaire et utiliser les AddEventListener

PHP

Pour moi php est le langage que je maitrise le plus. Au début de l'année nous apprenons avec MYSQL mais quand j'ai vue que c'étais obsolete je me suis tourné vers PDO et la seule chose ou j'ai beaucoup de mal à faire sont les fonctions

C#

C# est le langage que je maitrise le moins parmi les six. J'ai beaucoup de difficulté et même pour creer une simple classe j'ai beaucoup de mal mais avec le projet que je devais réaliser en deuxieme années j'ai un peux mieux compris le langage et les classes

Python

Python est un langage plutot facile en soit mais je ne peux pas me positionner dessus car je n'est jamais réaliser de projet ou autre sur python. Jai juste réaliser des exercices mais avec ce que j'ai eu au CCF de python je dirais que J'ai un niveau moyen

Mes projets réaliser

Projet Zoo

Projet Gacti

Pour ma Veille technologique j'ai décider de la réaliser sur l'intelligence artificielle

Qu'est ce que la veille technologique ?

La veille technologique consiste à s'informer de façon systématique sur les techniques les plus récentes et surtout sur leur mise à disposition commerciale.

Qu'est ce que l'intelligence Artificielle ?

L'intelligence artificielle est un « ensemble de théories et de techniques mises en œuvre en vue de réaliser des machines capables de simuler l'intelligence humaine ». Elle englobe donc un ensemble de concepts et de technologies, plus qu'une discipline autonome constituée
